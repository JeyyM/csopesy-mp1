INSTRUCTIONS ON HOW TO RUN THE PROGRAM:
----------------------------------------
1. Building and Running the Project:
     cd "c:\filepath\CSOPESY OS MP"
     cmake --build build
     .\build\csopesy_os_mp.exe

3. Using the Program:
   - The program will display a CSOPESY ASCII header and welcome message
   - You can enter the following commands:
   
   AVAILABLE COMMANDS:
   -------------------
   - initialize      : Load config.txt and initialize and use the system.
   - clear           : Clear the screen and reprint the header
   - exit            : Exit the program
   - scheduler-start : Currently prints mock data
   - scheduler-stop  : Currently prints mock data
   - report-util     : Currently prints mock data
   - screen -ls      : Currently prints mock data
   - screen -s <name>: Currently prints mock data
   - screen -r <name>: Currently prints mock data
